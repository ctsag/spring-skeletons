# Notes
This is a partial copy from the data-tests project. The intention is to
demonstrate how easy it is to swap Spring Data modules, going from JPA
on a relational H2 database to a directory database on MongoDB.

# Contents
This skeleton application involves the following areas :

## Spring Projects
* AssertJ
    - Assertions
* MongoDB
* Lombok
    - @Getter
    - @Setter
    - @ToString
* Spring Boot
    - @Autowired
* Spring Data JPA
    - Page
    - Pageable
    - PageRequest
    - PagingAndSortingRepository
    - Sort
* Spring Boot Testing
    - @SpringBootTest
    - @DataMongoTest

## Concepts
* AssertJ
* Spring Data JPA
    - Derived queries
    - Paging
    - Sorting
    - User defined methods
* Spring Data Mongo
